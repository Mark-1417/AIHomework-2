# AIHomework-2
## GRU 测试
使用了v2020版本的数据。

extract_sample.py是把原始数据和独立的表头文件捏到一块用的，方便预处理。其实和预处理一块做也行，因为这个文件原本是用来取10%数据的，但是我发现预处理全部数据也要不了太久，模型训练的时候再取10%就行。

preprocess_timeseries.py顾名思义是预处理用的，拼接metric和spec，再完成数据清洗，异常值修正，时序排序就可以直接用于模型训练啦。
